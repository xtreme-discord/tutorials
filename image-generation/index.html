Image Generation

1.What are Image Generation ?

It is the art of Manupulating Images in a better or funny way.


Image Generation in Xtreme includes features like manupulating images with GTA-V wasted screen and much more.

2. Getting Started

For getting help and a list of commands related to Image Generation, type x!help image
